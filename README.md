# Deutsch-Jozsa Quantum Algorithm Analysis

## Purpose
Explore the differences between classical and quantum approaches through five specific exercises.

## Setup Instructions
Run the following commands to set up and launch the notebook:

```bash
git clone https://github.com/PhatTanNguyen45/emerging_technologies
cd emerging_technologies
pip install -r requirements.txt
jupyter notebook
```

## Folder Organization
- `data/`: Stores data used in exercises and experiments.
- `img/`: Stores illustrative images, charts, or notebook outputs.
